Mijn Opleiding

Ik ben een software developer student bij ROC Mondriaan. Als software developer ontwikkel ik voornamelijk zelfstandig (onderdelen van) software, programma's of websites. Ik zorg ervoor dat ze werken. Het gaat niet alleen om het maken van nieuwe dingen. Ik zorg er ook voor dat bestaande software blijft werken. Ik leer in de opleiding ook hoe ik goed kan doorvragen wat de klant (eigenlijk) nodig heeft. Heb ik de wensen goed op een rij dan kan het ontwerpen en ontwikkelen gaan beginnen. Is de applicatie klaar? Dan help ik, na het uitvoerig testen van de software, ook mee aan de invoering ervan.
